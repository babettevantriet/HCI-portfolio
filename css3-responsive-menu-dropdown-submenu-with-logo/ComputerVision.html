Assignment 1

Het is de bedoeling om een programma te schrijven waarin er drie verschillende figuren staan die:
- Op een verschillende locatie staan
- Allemaal een andere kleur hebben
- Verschillende lengte en breedte hebben
- En verschil in wel of geen border.

De stoke() of noStroke() geeft aan of er wel of geen border omheen zit.
fil(), houd in met welke kleur het ovaal ingekleurd wordt.

ellipsemode(CENTER) betekend dat hij de afstand berekend vanaf het midden van het ovaal.
Tot slot heb je de elipse(‘getal1’, ‘getal2’, ‘getal3’, ‘getal4’)
- Getal1 gaat over de x-as
- Getal2 gaat over de y-as
- Getal3 gaat over de breedte
- Getal4 gaat over de lengte

Assignment CV 1

Voor deze opdracht ga je aan de slag met de live webcam. In het begin leer je hoe je bibliotheken moet importeren en hoe je ze daarna kunt aanpassen en gebruiken. Voor deze opdracht moest je gebruik maken van gezichtsherkenning. Hierbij kreeg je een vierkant, en daarbij was het de opdracht om het vierkant rond te maken. Het was niet zo moeilijk om het vierkant rond te krijgen, wel was het nog een kunst om daarna de circle goed op mn gezicht te krijgen.

import gab.opencv.*;
import processing.video.*;
import java.awt.*;

Capture video;
OpenCV opencv;

void setup() {
size(640, 480);
video = new Capture(this, 640/2, 480/2);
opencv = new OpenCV(this, 640/2, 480/2);
opencv.loadCascade(OpenCV.CASCADE_FRONTALFACE);

video.start();
}

void draw() {
scale(2);
opencv.loadImage(video);

image(video, 0, 0 );

stroke(0, 255, 255);
noFill();
Rectangle[] faces = opencv.detect();
println(faces.length);

for (int i = 0; i < faces.length; i++) {
println(faces[i].x + "," + faces[i].y);
ellipseMode(CORNER);
ellipse(faces[i].x, faces[i].y, faces[i].width, faces[i].height);
}
}

void captureEvent(Capture c) {
c.read();
}

Assignment AR

Deze opdracht is een opdracht voor augmented reality. Ook hier moest ik weer een bibliotheek downloaden. Hierna heb ik de 'marker' gedownload voor op mijn telefoon en uiteindelijk moest ik ervoor zorgen dat er een vierkant via de webcam op de marker kwam. Op de afbeelding is te zien dat dit niet optimaal gelukt is, dit kwam doordat de marker op mijn telefoon weergeven was en deze had veel weerspiegeling waardoor hij niet goed te lezen was.

/**

*
* This sketch is shortest sample.
* The sketch shows cube on the marker of "patt.hiro".
* Any pattern and configuration files are found in libraries/nyar4psg/data inside your sketchbook folder.
*/
import processing.video.*;
import jp.nyatla.nyar4psg.*;

Capture cam;
MultiMarker nya;

void setup() {
size(640,480,P3D);
colorMode(RGB, 100);
println(MultiMarker.VERSION);
cam=new Capture(this,640,480);
nya=new MultiMarker(this,width,height,"../../data/camera_para.dat",NyAR4PsgConfig.CONFIG_PSG);
nya.addARMarker("../../data/patt.hiro",80);
cam.start();
}

void draw()
{
if (cam.available() !=true) {
return;
}
cam.read();
nya.detect(cam);
background(0);
nya.drawBackground(cam);//frustumを考慮した背景描画
if((!nya.isExist(0))){
return;
}
nya.beginTransform(0);
fill(0,0,255);
translate(0,0,20);
box(40);
nya.endTransform();
}
